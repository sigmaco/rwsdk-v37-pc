--------------------------------------------------------------------------
                  PVS Editor for RenderWare Graphics
--------------------------------------------------------------------------

The PVS editor is a tool that allows the PVS data in a world to be edited.

The PVS Editor document, pvsedit.pdf, can be found in 
docs\tools in the RenderWare Graphics directory.

The pvsedit.chm included in this directory also contains information
about the PVS Editor.
