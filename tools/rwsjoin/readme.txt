--------------------------------------------------------------------------
                 File Joining Tool for RenderWare Graphics
--------------------------------------------------------------------------

rwstool
rwsjoin is used to join two or more RenderWare binary stream files into
a single file.

The tool's option is

Add : -a <src>

Adds the specified source file to the destination file. You can add more
than one file by prefixing each input file with the -a option.
